// 01 — ОБО МНЕ

Обо мне

Собираю "дорогой" UI, верстаю по макету и делаю адаптив.

Figma → HTML/CSS

Я студент и начинающий веб-разработчик. Собираю портфолио и делаю современные лендинги. Люблю чистую вёрстку и когда UI выглядит "дорого": ровная сетка, отступы, аккуратная типографика. Умею повторять дизайн из Figma и доводить до идеала на разных экранах.

Мой фокус — лендинги, карточки, формы, секции, адаптив и микро-анимации. Люблю собирать компоненты как в дизайн-системе: единые кнопки, бейджи, отступы, состояния.

Делаю так: беру макет → собираю структуру → сетка (Grid/Flex) → типографика → состояния/hover → адаптив. Если нужно, быстро правлю UI: выравниваю, ускоряю загрузку, убираю "прыжки" и делаю аккуратнее.

Сейчас прокачиваю CSS (Grid/Flex), базовый JavaScript и GitHub-flow. Готов брать мелкие задачи: правки, верстка секций, формы, адаптив под мобилку.
